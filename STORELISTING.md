WhisperShroom nutzt die .NET Desktop Runtime und die OpenAI Whisper API für Spracherkennung.
Eine schlanke Voice-to-Text App für Windows 11. Drücke einen globalen Hotkey, sprich, und der transkribierte Text landet direkt in deiner Zwischenablage — bereit zum Einfügen in jede App.

Unter der Haube nutzt WhisperShroom die OpenAI Whisper API, die für ihre exzellente Spracherkennung bekannt ist. Die App läuft unauffällig im System Tray und ist sofort einsatzbereit.

Perfekt für alle, die schnell diktieren wollen — in Chat-Fenster, E-Mails, Editoren, Browser-Formulare oder jede andere Anwendung.

Voraussetzung: Ein eigener OpenAI API-Key (kostenlos erstellbar unter platform.openai.com). Die Kosten für die Whisper API sind minimal (ca. $0.006 pro Minute).

Open Source: github.com/shroomlife/whisper-shroom
